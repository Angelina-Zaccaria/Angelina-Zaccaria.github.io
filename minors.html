Software Engineering minor (for non-Software Engineering majors only)

Students must possess a cumulative GPA of at least 2.7 at graduation. A Software Engineering minor requires a two-course overload. The following are required courses:

  • MA 134 Discrete Math or CS 135 Discrete Structures
  • SSW 215 Individual Software Engineering
  • SSW 315 Object Based Software Engineering
  • SSW 345 Model Based Software Engineering
  • SSW 555 Agile Methods
  • SSW 581 System Engineering Architecture

The following prerequisite is needed to undertake the minor program:

  • E115 Introduction to Programming or CS115 Introduction to Computer Science
